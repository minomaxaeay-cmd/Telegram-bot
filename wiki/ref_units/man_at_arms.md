# Man-at-Arms

The reliable line [infantry](../units.md) — a balanced generalist that anchors
most armies. Requires a [Barracks](../ref_buildings/barracks.md).

## Stats
| Stat | Value |
|---|---|
| Cost | 130 gold |
| HP | 27 |
| Soft attack | 15 |
| Hard attack | 2 |
| Piercing | 1 |
| Armor | 2 |
| Hardness (`hard`) | 0.2 |
| Width | 2 |
| Upkeep | 0.5 gold/min |
| Combat valid | Yes |

These values are generated by the parametric backbone (`game_balance.py`) — no
exceptions; the Man-at-Arms is the "on-formula" generalist.

## Role & counters
- Gate: requires [Barracks](../ref_buildings/barracks.md) Lv1.
- **Beats [Peasants](peasant.md) and [Pikemen](pikeman.md)** (high soft attack).
- **Loses to [Knights](knight.md)** (heavy armor + hard hits).
- Strong field win-rate (**~67%** per the offline `balance_sim.py` simulator, not a
  source constant) — one of the two generalist pillars alongside the Knight.
